# Zerodha Clone – Full Stack Web Application

A full-stack Zerodha clone built using **React**, **Node.js**, **Express**, and **MongoDB Atlas**.  
This project includes a complete frontend website, backend REST API, and an admin dashboard — all inside one repository.

---

## ✨ Features

- Modern Zerodha-style UI  
- Pages: Home, About, Pricing, Products, Signup, Support  
- Admin dashboard for UI visualization  
- MongoDB database integration  
- Express REST API  
- Secure environment variables  
- Axios-based communication  
- Full deployment setup (Netlify + Render)
